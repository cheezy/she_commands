## Theme

| Property | Value |
|---|---|
| Color Mode | Light |
| Color Variant | Monochrome |
| Primary Color | `#000000` |
| Neutral Color | `#ffffff` |
| Headline Font | Space Grotesk |
| Body Font | Inter |
| Label Font | Inter |
| Roundness | Round 4 (sharp/minimal) |

## Principles

- Stark black-and-white palette with maximum contrast
- Bold typography as the primary design element
- Generous whitespace and spacious layouts
- Sharp, clean edges — no decorative curves
- Professional, commanding visual presence
- Photography-forward with strategic image placement

## Color Usage

- Primary: Black (#000000) for headlines, buttons, and strong elements
- Background: White (#FFFFFF) for clean, open layouts
- Gray tones for subtle hierarchy and dividers
- Strictly monochromatic — no vibrant accent colors

## Typography

- Headlines: Bold, geometric sans-serif — large and commanding
- Body: Clean, readable sans-serif
- Generous line height and letter spacing for premium feel

## Layout

- Grid-based with generous padding
- High-impact sections with full-width imagery
- Animated marquee-style text elements ("Power starts here", "Execute relentlessly")
